El proyecto de Checker es un proyecto web el cual tiene como objetivo ayudar en la revision del empacado de productos de una empresa de logistica, esto para cumplir con los estandares de calidad de la misma.

# Tecnologias usadas

El proyecto esta basado en una estructura MEAN, el cual contiene las siguientes tecnologias:
- MongoDB (para la base de datos)
- Express
- Angular
- NodeJS

# Instalacion

Para la instalacion e inicializacion del proyecto, seguir los siguientes pasos:

1. Clonar el repositorio con **Git CLI** o descargando directamente desde el repo.
2. Abrir el proyecto **Checker** en un editor de codigo (Visual Studio Code recomendado).
3. Abrir dos terminales y viajar a las carpetas de ```Front``` y ```Back``` respectivamente.
4. Instalar dependencias con ```npm i```.
5. Iniciar ```Front``` con el comando ```ng serve``` de **Angular**.
6. Iniciar ```Back``` con el comando ```npm start``` de **Node**.
